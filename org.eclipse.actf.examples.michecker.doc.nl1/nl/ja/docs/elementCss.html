ID/AccessKey/Class/CSS Inspector (確認) パネル

ID/AccessKey/Class/CSS Inspector パネルでは、HTML 要素に付与されたID、AccessKey、Class の各属性、および対象ウェブページに適用されているスタイルシート (CSS) の情報を確認することができます。 このパネルには4つのタブがあります。

  • ID タブ
  • AccessKey タブ
  • Class タブ
  • CSS タブ

ID タブ

ID タブには、ID 属性が付与されている HTML 要素が表示されます。 ID属性は、スタイルシートを適用する HTML 要素を指定するために用いられます。

  • Value: ID 属性の値です。
  • Tag Name: そのID 属性が付与されている HTML 要素のタグ名です。

AccessKey タブ

AccessKey タブには、AccessKey 属性が付与されている HTML 要素が表示されます。 AccessKey は単一の文字を表すキーで、ALT キーと同時に押すことにより、その AccessKey 属性が付与されている HTML 要素にフォーカスが与えられます。

  • Value: AccessKey 属性の値、つまり当該 HTML 要素に与えられたアクセスキー名です。
  • Tag Name: その AccessKey 属性が付与されている HTML 要素のタグ名です。

Class タブ

Class タブには、Class 属性が付与されている HTML 要素が表示されます。 Class 属性も ID 属性と同様に、スタイルシートを適用する HTML 要素を指定するために用いられます。

  • Value: Class 属性の値です。
  • Tag Name: その Class 属性が付与されているHTML要素のタグ名です。
<ヒント>
IDタブ、AccessKey タブ、Class タブの各行は選択できます。 選択することにより、対応する要素がブラウザビュー上および視覚化ビュー上でハイライトされます。 また、これらのタブの表示内容は項目見出し（Value または TagName）を選択することにより並べ替えることができます。

CSS タブ

CSS タブには、対象ウェブページに適用されているスタイルシート (CSS) が表示されます。
