Installation guide to use ACTF Visualization in Web/J2EE/PHP perspectives

  1. Launch Eclipse Install dialog from Help -> Install New Software -> Add menu.
  2. Select Galileo update site (http://download.eclipse.org/releases/galileo). (see Figure 1)
  3. Select ACTF Visualization Extension for PDT Feature (for PHP perspective) and ACTF Visualization Extension for WST Feature (for Web/J2EE perspective) under General Purpose Tools category.
  4. Select Next and follow installation wizard.
