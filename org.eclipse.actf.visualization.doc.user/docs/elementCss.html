ID/AccessKey/Class/CSS Inspector Panel

The ID/AccessKey/Class/CSS Inspector Panel shows information on the attributes (ID, AccessKey, and Class) of elements that have been set and the information for any Cascading Style Sheets (CSS) applied to the target page. This panel has four tabs.

ID tab

The ID tab lists the elements that have ID attributes. The ID attribute of an HTML element can be used by Web browsers to determine which style sheets to apply to the element.

  • Value: The value of the ID attribute.
  • Tag Name: The tag name of the HTML element.

AccessKey tab

The AccessKey tab lists the elements that have AccessKey attributes. An AccessKey attribute of an HTML element assigns an access key to the element. An access key is a single character, and pressing the ALT key plus an access key moves the focus to the corresponding element.

  • Value: The value of the AccessKey attribute, which is the access key assigned to the element.
  • Tag Name: The tag name of the HTML element.

Class tab

The Class tab lists the elements that have Class attributes:

  • Value: The value of the Class attribute.
  • Tag Name: The tag name of the HTML element.
Each line in the ID, AccessKey, and the Class tabs can be clicked. When a line is clicked, its corresponding element is highlighted, both in the Visualization view and the ACTF Browser Editor. The lines can also be sorted. When a column header is clicked, all of the lines in the table are sorted by the values in that column.

CSS tab

The CSS tab lists the URLs of the CSS files that are applied to the target page.
