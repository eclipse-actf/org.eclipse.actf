Installation Guide for OpenDocument Accessibility function

Step 1

  • Install OpenOffice.org 3.x (later than 3.0.1). Launch OpenOffice.org and finish initialization.

Step 2

  • Launch this application.

Step 3

  • If this installation guide does not appear when switching to OpenDocument Accessibility perspective, then the OpenDocument function has been installed successfully.
