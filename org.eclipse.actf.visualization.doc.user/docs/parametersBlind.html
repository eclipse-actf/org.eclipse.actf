Blind Usability Visualization preference page

The Blind Usability Visualization preference page is used to set the Visualization mode and language that ACTF Blind Visualization uses.

Visualization mode: Select either Voice browser output mode or Layout mode.

  • Voice browser output mode: Selecting this option causes the output text from a voice browser to be displayed in the Visualization View (rather than being read aloud). In the current version of ACTF, only Home Page Reader can be used as the voice browser.

  • Layout mode: Specifies the colors and times used to show reaching times, tag colors, and table structure colors in the Visualization View.

    • Select Visualize reaching time to display the reaching time from the top of the page to each element in the Visualization View. By default, black indicates that it takes more than 90 seconds to get to that region. The reaching time and display color are controlled here.

    • Selecting Colorize accessible tags causes table headers, heading tags, form elements, and label tags to be colored in the Visualization View. The colors can be set by clicking the buttons that follow the tag names.

    • Selecting Visualize table structure causes the borders of table cells to be drawn with black lines. The colors of the lines can be set by clicking the adjacent button.

    • Set default colors: Clicking this button cancels the current color settings and resets the default colors for these options.

Inappropriate Alt: Clicking Edit Inappropriate Alt List button allows editing the list of inappropriate ALT-attributes. When an ALT-attribute of an element in the target page is contained in this list, the ALT-attribute will be detected as a problem.

Help: Click this button to open this page.
Apply: Click this button to save the current settings.
OK: Click this button to save the current settings and to close the dialog box.
Cancel: Click this button to close the dialog box without changing the settings.
