OpenDocument Accessibility perspective configurations

OpenDocument Format Checker Setting

Users can change the OpenDocument Format checker settings using this panel (Figure 1). Some editors do not save the correct ODF version information in the files. If this option is selected, ACTF automatically detects the correct version by examining the ODF file content, and checks the accessibility as the file is created in the detected ODF version.
