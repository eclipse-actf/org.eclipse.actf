HTML Source viewer

The HTML Source viewer shows the content of the HTML document of the target page.

  • Click View Source HTML in the context menu of a line in the Line View Pane (Blind Mode).
When you open the HTML Source viewer from the Line view in the blind mode, the HTML element corresponding to the selected problem is highlighted.


The HTML Source viewer also shows the line and column numbers of the highlighted element or of the cursor position in the HTML document. This makes it easy to find the locations in the HTML document that need to be repaired.

Click the Search button to open the Search dialog box to look for a text string within the HTML document.
