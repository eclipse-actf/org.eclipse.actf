Using ACTF Visualization in Web/J2EE/PHP perspectives

ACTF provides extension features for WST/PDT. If you install these features, you can use accessibility visualization functions of ACTF in Web, J2EE and PHP perspectives. After opening target content by using 'Run As' menu ('Run on Server', 'PHP Script', 'PHP Web Page'), select Visualization menu to confirm usability of blind/lowvision users.
